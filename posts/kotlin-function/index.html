Kotlinの関数について学習します。

第一級オブジェクト

「第一級オブジェクト(first-class object)」とは、以下の性質を持つオブジェクトのことを指します。

  • 変数に代入することができる
  • 関数の引数として利用できる
  • 関数の結果として返すことができる

そして、Kotlinでは、関数を第一級オブジェクトとして扱うことができます。

既に定義されている関数の名前の頭に::を記載することで、その関数の情報を取得できます。

fun addOne(i: Int): Int = i + 1

fun main(args: Array<String>) {
    println(::addOne)
    // 「fun addOne(kotlin.Int): kotlin.Int」と出力される
}

また、取得した情報を変数に代入することが可能です。

fun addOne(i: Int): Int = i + 1

fun main(args: Array<String>) {
    val functionObj = ::addOne
    println(functionObj(10))
    // 「11」と出力される
}

関数型

::で取得した情報にも型が存在しており、上記の処理では型推論に任せて、型の指定を省略しました。

型を明示的に指定する場合、以下のように記述します。

1

(1番目の引数の型, 2番目の引数の型, ...) -> 返り値の型

高階関数

関数を引数として受け取ったり、返り値として返すような関数のことを、「高階関数(higher-order function)」と呼びます。

ここに、最初にAが存在する位置を返す関数と最初に大文字が出現する位置を返す関数が記述されています。

// 最初にAが存在する位置を返す関数
fun firstA(str: String): Int {
    tailrec fun go(str: String, index: Int): Int =
        when {
            str.isEmpty() -> -1
            str.first() == 'A' -> index
            else -> go(str.drop(1), index + 1)
        }
    return go(str, 0)
}

// 最初に大文字が出現する位置を返す関数
fun firstUpperCase(str: String): Int {
    tailrec fun go(str: String, index: Int): Int =
        when {
            str.isEmpty() -> -1
            str.first().isUpperCase() -> index
            else -> go(str.drop(1), index + 1)
        }
    return go(str, 0)
}

firtsAとfirstUpperCaseを比べると、whenの中のindexを返す条件以外は共通であることがわかります。

// 共通処理
fun first(str: String, predicate: (Char) -> Boolean): Int {
    tailrec fun go(str: String, index: Int): Int =
        when {
            str.isEmpty() -> -1
            predicate(str.first()) -> index
            else -> go(str.drop(1), index + 1)
        }
    return go(str, 0)
}

// 最初にAが存在する位置を返す関数
fun firstA(str: String): Int {
    fun isA(c: Char): Boolean = c == 'A'
    return first(str, ::isA)
}

// 最初に大文字が出現する位置を返す関数
fun firstUpperCase(str: String): Int {
    fun isUpperCase(c: Char): Boolean = c.isUpperCase()
    return first(str, ::isUpperCase)
}

indexを返す条件を「文字を引数に取って、Booleanを返す関数」にし、predicateという名前の関数として、firstの引数として与えました。

関数を直接引数に使用することによって、上記のような処理が実装できます。

ラムダ式

Kotlinのラムダ式は型推論が機能します。 型指定の省略方法は以下の2通りです。

1
2
3
4
5
6
7

val square1 = { i: Int ->
    i * i
}

val square2: (Int) -> Int = { i ->
    i * i
}

また、ラムダ式の引数は1つの場合、変数itを使用できます。

1
2
3

val square2: (Int) -> Int = {
    it * it
}
